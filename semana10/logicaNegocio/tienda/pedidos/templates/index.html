📦 API de Pedidos

Documentación de endpoints

URL: http://localhost:8000/api/

Crear Pedido

Endpoint: /api/crear/

Método: POST

Descripción: Crea un nuevo pedido a partir de una lista de items con su precio y cantidad y calcula el total.

Body (JSON):
 
                            [
                                {
                                    "precio":26,
                                    "cantidad":5
                                }
                            ]
Respuesta:
                            {
                                "id": 3,
                                "fecha": "2026-03-28T01:42:33.825843Z",
                                "total": 75.0
                            }
